# Spiritual Knowledge Base

A modern, accessible collection of spiritual wisdom inspired by Krishna’s teachings in the Bhagavad Gita. This project transforms ancient lessons into practical guides for overcoming self-doubt, finding purpose, and living with courage in today’s world.

## Project Structure
- `krishna_life_lession/` — Chapters rewritten for a general audience, with actionable advice, examples, and image prompts.
- `processed_articles/` — Source articles and drafts for content transformation.
- `images/` — Visual assets and AI-generated illustrations for each chapter.
- `.gitignore` — Excludes cache, environment, images, and other non-essential files from version control.

## Features
- **Accessible Chapters:** Clear, engaging, and practical spiritual lessons.
- **Image Prompts & Illustrations:** Each section includes descriptive prompts and corresponding images.
- **Modern Parallels:** Real-life examples and exercises connect ancient wisdom to everyday challenges.
- **Ready for Publishing:** Content is formatted for GitHub Pages or other static site generators.

## How to Use
1. **Read the Chapters:** Start with `krishna_life_lession/chapter2.md` for Krishna’s battlefield mindset.
2. **View Images:** Each section features an image prompt and illustration.
3. **Publish Online:** Use GitHub Pages or similar tools to share the content as a multi-chapter website.
4. **Customize:** Expand chapters, add new lessons, or update image prompts as needed.

## Getting Started
1. Clone the repository:
   ```sh
   git clone <your-repo-url>
   ```
2. Edit or add chapters in the `krishna_life_lession` folder.
3. Generate or update images in the `images` folder.
4. Publish using GitHub Pages or your preferred platform.
